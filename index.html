NERDWERDS

NERDWERDS was a game created during an 'educational tool' themed hackathon. Its purpose is to help beginner coders with term recognition. The user drags and drops words into labeled buckets and is scored based on performance.

  • Technologies used: HTML, CSS, Javascript, Node, Express, JSON, Git
  • Use of JQuery word cloud and drag and drop
  • As team leader, I delegated tasks and made final decisions
